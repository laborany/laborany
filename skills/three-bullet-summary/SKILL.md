# 三点摘要

将任意输入文本精炼为恰好三个要点，每个要点一句话。

## 工作流程

1. 通读全文，识别核心论点与关键信息
2. 按重要性排序，提取最关键的三个独立观点
3. 每个要点压缩为一句简洁、完整、可独立理解的陈述

## 输出格式

```
- [要点一]
- [要点二]
- [要点三]
```

## 规则

- 恰好三条，不多不少
- 每条一句话，不超过 30 字
- 保留原文核心含义，不添加推测
- 去除冗余修饰，只留干货
- 如原文信息不足三点，从不同角度拆分主论点
