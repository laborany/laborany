# 研究框架模板集合

本文档包含不同研究类型对应的研究框架，用于在生成提示词时根据研究类型选择合适的框架。

---

## 1. 时间线框架（事件/发言整理）

**适用场景：**
- 整理某次会议的发言
- 梳理某事件的发展过程
- 按时间顺序呈现历史信息

**框架内容：**
```
请按照以下时间线框架进行研究：

### 第一阶段：事件背景
- 时间范围：[具体时间]
- 主要背景：[描述]
- 关键参与者：[列表]

### 第二阶段：时间线梳理
按时间顺序整理事件/发言：

时间 | 事件/内容 | 来源 | 备注
-----|----------|------|-----
[时间1] | [内容] | [来源] | [备注]
[时间2] | [内容] | [来源] | [备注]
...

### 第三阶段：主题分类
将内容按主题归类：

主题1：[主题名称]
- 要点1：[具体内容]
- 要点2：[具体内容]

主题2：[主题名称]
- 要点1：[具体内容]
- 要点2：[具体内容]

### 第四阶段：关键信息提取
- 核心观点/事实：[列表]
- 重要数据：[列表]
- 直接引用：[关键引言]
```

## 2. PEST分析框架（趋势分析）

**适用场景：**
- 行业发展趋势分析
- 宏观环境分析
- 政策影响评估

**框架内容：**
```
请按照PEST框架进行分析：

### P - 政治环境 (Political)
- 相关政策法规：[列举]
- 监管动态：[描述]
- 政府态度/支持：[说明]
- 国际政治因素：[如有]

### E - 经济环境 (Economic)
- 市场规模：[数据]
- 投资趋势：[描述]
- 经济影响：[分析]
- 成本/收益考量：[说明]

### S - 社会环境 (Social)
- 用户接受度：[数据/现象]
- 社会影响：[正面/负面]
- 文化因素：[说明]
- 公众关注点：[列举]

### T - 技术环境 (Technological)
- 技术突破：[列举]
- 创新方向：[描述]
- 技术挑战：[分析]
- 发展瓶颈：[说明]

### 综合分析
基于以上四个维度，总结：
- 主要机遇：[列表]
- 主要挑战：[列表]
- 发展预期：[判断]
```

## 3. SWOT分析框架（战略分析）

**适用场景：**
- 公司/组织分析
- 竞争对手分析
- 项目评估

**框架内容：**
```
请按照SWOT框架进行分析：

### S - 优势 (Strengths)
- 核心竞争力：[列举]
- 资源优势：[说明]
- 市场地位：[描述]
- 其他优势：[补充]

### W - 劣势 (Weaknesses)
- 主要短板：[列举]
- 资源限制：[说明]
- 市场挑战：[描述]
- 其他劣势：[补充]

### O - 机遇 (Opportunities)
- 市场机遇：[列举]
- 政策红利：[说明]
- 技术机遇：[描述]
- 其他机遇：[补充]

### T - 威胁 (Threats)
- 竞争压力：[列举]
- 政策风险：[说明]
- 技术颠覆：[描述]
- 其他威胁：[补充]

### 战略建议
基于SWOT分析：
- SO策略（发挥优势抓住机遇）：[建议]
- WO策略（弥补劣势抓住机遇）：[建议]
- ST策略（发挥优势应对威胁）：[建议]
- WT策略（减少劣势应对威胁）：[建议]
```

## 4. 多维度对比框架（观点整理）

**适用场景：**
- 整理多方观点
- 对比不同立场
- 争议问题分析

**框架内容：**
```
请按照以下多维度对比框架整理观点：

### 观点分类
将观点按立场分类：

#### 支持方观点
- 观点1：[描述]
  - 代表人物/机构：[列表]
  - 核心论据：[说明]
  - 支撑数据/案例：[列举]

- 观点2：[描述]
  - 代表人物/机构：[列表]
  - 核心论据：[说明]
  - 支撑数据/案例：[列举]

#### 反对方观点
- 观点1：[描述]
  - 代表人物/机构：[列表]
  - 核心论据：[说明]
  - 支撑数据/案例：[列举]

#### 中立/审慎观点
- 观点1：[描述]
  - 代表人物/机构：[列表]
  - 核心论据：[说明]
  - 支撑数据/案例：[列举]

### 争议焦点
列出各方的主要分歧点：
- 焦点1：[描述分歧]
- 焦点2：[描述分歧]
- 焦点3：[描述分歧]

### 共识领域
列出各方的共同观点：
- 共识1：[描述]
- 共识2：[描述]

### 观点演进
说明观点随时间的变化：
- 早期观点：[描述]
- 当前变化：[说明]
- 趋势判断：[分析]
```

## 5. 商业模式分析框架（企业研究）

**适用场景：**
- 研究某公司的商业模式
- 投资分析
- 竞争分析

**框架内容：**
```
请按照以下商业模式分析框架进行研究：

### 公司概况
- 基本信息：[成立时间、规模、业务范围]
- 发展历程：[关键里程碑]
- 市场地位：[行业排名、市场份额]

### 商业模式画布
#### 1. 价值主张
- 核心价值：[描述]
- 解决什么问题：[说明]
- 与竞争对手的差异化：[对比]

#### 2. 目标客户
- 客户群体：[细分]
- 客户需求：[分析]

#### 3. 渠道通路
- 销售渠道：[列举]
- 分销方式：[说明]

#### 4. 客户关系
- 获客方式：[描述]
- 留存策略：[说明]

#### 5. 收入来源
- 收入模式：[列举]
- 定价策略：[说明]

#### 6. 核心资源
- 关键资源：[列举]

#### 7. 关键业务
- 核心业务：[列举]

#### 8. 重要合作
- 合作伙伴：[列举]
- 供应链：[描述]

#### 9. 成本结构
- 主要成本：[列举]

### 财务表现
- 收入规模：[数据]
- 盈利状况：[分析]
- 增长趋势：[说明]

### 竞争分析
- 主要竞争对手：[列举]
- 竞争优势：[分析]
- 潜在威胁：[说明]

### 发展前景
- 增长机会：[列举]
- 风险挑战：[分析]
- 未来展望：[判断]
```

## 6. 概览+重点深入框架（探索了解）

**适用场景：**
- 用户没有明确方向
- 需要全面了解某个主题
- 探索性研究

**框架内容：**
```
请按照以下探索框架进行研究：

### 总体概览
- 基本概念：[定义]
- 发展现状：[描述]
- 主要方向/领域：[列举]
- 重要意义：[说明]

### 重点领域深入
选择3-5个最重要的领域进行深入研究：

#### 领域1：[名称]
- 简介：[概述]
- 最新进展：[列举]
- 关键突破/事件：[描述]
- 代表人物/机构：[列表]

#### 领域2：[名称]
- 简介：[概述]
- 最新进展：[列举]
- 关键突破/事件：[描述]
- 代表人物/机构：[列表]

[继续其他领域...]

### 重要人物/机构
列出该领域的领先者：
- 人物：
  1. [姓名] - [贡献/地位]
  2. [姓名] - [贡献/地位]
- 机构：
  1. [机构名] - [贡献/地位]
  2. [机构名] - [贡献/地位]

### 发展时间线
- [时间]：[关键事件]
- [时间]：[关键事件]
- [时间]：[关键事件]

### 当前热点
列举当前最受关注的话题：
1. [热点1]：[描述]
2. [热点2]：[描述]
3. [热点3]：[描述]

### 延伸了解建议
推荐深入探索的方向：
- 方向1：[说明原因]
- 方向2：[说明原因]
- 方向3：[说明原因]
```

## 框架选择指南

| 研究类型 | 推荐框架 | 备选框架 |
|---------|---------|---------|
| 事件/发言整理 | 时间线框架 | 主题分类框架 |
| 趋势分析 | PEST框架 | SWOT框架 |
| 观点整理 | 多维度对比框架 | - |
| 企业/公司研究 | 商业模式分析框架 | SWOT框架 |
| 战略分析 | SWOT框架 | PEST框架 |
| 探索了解 | 概览+重点深入框架 | - |
