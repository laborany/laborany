## 使用说明

### 组装提示词的步骤

1. **确定研究类型** → 选择研究框架
2. **拆解子问题** → 列出3-5个核心问题
3. **定义信息源** → 明确信息来源要求
4. **选择详细程度** → 选择输出格式
5. **选择目标平台** → 应用平台特定优化
6. **组装完整提示词** → 使用对应模板

### 模板变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| {研究主题} | 简洁的研究题目 | "AI行业2026发展趋势" |
| {研究目标} | 详细的研究目标描述 | "全面分析AI行业的技术、商业、政策发展趋势..." |
| {研究框架} | 选定的框架内容 | 时间线框架/PEST框架等 |
| {子问题} | 具体的研究问题 | "AI技术有哪些突破性进展？" |
| {信息源类型} | 优先的信息源类型 | "官方发布、权威媒体" |
| {时间范围} | 信息时效要求 | "2025-2026年" |
| {详细程度} | 简洁/标准/完整 | 根据用户需求选择 |
