# 技术指标分析参考

股票技术分析常用指标的计算方法与信号解读。

## 移动平均线 (MA)

### 计算方法
```
MA(N) = (C1 + C2 + ... + CN) / N
其中 C 为收盘价，N 为周期
```

### 常用周期
| 周期 | 用途 |
|-----|------|
| MA5 | 短期趋势，敏感度高 |
| MA10 | 短期趋势确认 |
| MA20 | 中期趋势，月线参考 |
| MA60 | 中期趋势，季线参考 |
| MA120 | 长期趋势，半年线 |
| MA250 | 长期趋势，年线 |

### 信号解读
- **多头排列**: MA5 > MA10 > MA20 > MA60 → 强势上涨
- **空头排列**: MA5 < MA10 < MA20 < MA60 → 弱势下跌
- **金叉**: 短期MA上穿长期MA → 买入信号
- **死叉**: 短期MA下穿长期MA → 卖出信号

## MACD (指数平滑异同移动平均线)

### 计算方法
```
EMA12 = 前一日EMA12 × 11/13 + 今日收盘价 × 2/13
EMA26 = 前一日EMA26 × 25/27 + 今日收盘价 × 2/27
DIF = EMA12 - EMA26
DEA = 前一日DEA × 8/10 + 今日DIF × 2/10
MACD柱 = (DIF - DEA) × 2
```

### 信号解读
| 信号 | 条件 | 含义 |
|-----|------|------|
| 金叉 | DIF上穿DEA | 买入信号 |
| 死叉 | DIF下穿DEA | 卖出信号 |
| 零轴上方金叉 | DIF>0 且金叉 | 强势买入 |
| 零轴下方死叉 | DIF<0 且死叉 | 强势卖出 |
| 顶背离 | 价格新高，MACD未新高 | 见顶信号 |
| 底背离 | 价格新低，MACD未新低 | 见底信号 |

## RSI (相对强弱指标)

### 计算方法
```
RS = N日内上涨总幅度平均值 / N日内下跌总幅度平均值
RSI = 100 - 100/(1+RS)
常用周期: RSI6, RSI12, RSI24
```

### 信号解读
| 区间 | 含义 | 操作建议 |
|-----|------|---------|
| RSI > 80 | 超买区 | 考虑卖出 |
| RSI 50-80 | 强势区 | 持有或买入 |
| RSI 20-50 | 弱势区 | 观望或减仓 |
| RSI < 20 | 超卖区 | 考虑买入 |

### 背离信号
- **顶背离**: 价格创新高，RSI未创新高 → 卖出信号
- **底背离**: 价格创新低，RSI未创新低 → 买入信号

---

## KDJ (随机指标)

### 计算方法
```
RSV = (收盘价 - N日最低价) / (N日最高价 - N日最低价) × 100
K = 2/3 × 前一日K + 1/3 × RSV
D = 2/3 × 前一日D + 1/3 × K
J = 3K - 2D
常用周期: N=9
```

### 信号解读
| 信号 | 条件 | 含义 |
|-----|------|------|
| 金叉 | K上穿D | 买入信号 |
| 死叉 | K下穿D | 卖出信号 |
| 超买 | K,D > 80 | 高位风险 |
| 超卖 | K,D < 20 | 低位机会 |
| J值钝化 | J > 100 或 J < 0 | 极端状态 |

## 布林带 (BOLL)

### 计算方法
```
中轨 = MA(N)  (通常N=20)
上轨 = 中轨 + 2 × 标准差
下轨 = 中轨 - 2 × 标准差
```

### 信号解读
| 位置 | 含义 |
|-----|------|
| 价格触及上轨 | 短期压力位，可能回调 |
| 价格触及下轨 | 短期支撑位，可能反弹 |
| 价格沿上轨运行 | 强势上涨趋势 |
| 价格沿下轨运行 | 弱势下跌趋势 |
| 带宽收窄 | 变盘信号，即将突破 |
| 带宽扩大 | 趋势加速 |

## 成交量分析

### 量价关系
| 价格 | 成交量 | 含义 |
|-----|-------|------|
| 上涨 | 放量 | 健康上涨，趋势延续 |
| 上涨 | 缩量 | 上涨乏力，警惕回调 |
| 下跌 | 放量 | 恐慌抛售，可能加速 |
| 下跌 | 缩量 | 抛压减轻，可能企稳 |

### 特殊形态
- **天量天价**: 成交量创新高伴随价格新高 → 见顶信号
- **地量地价**: 成交量极度萎缩 → 可能见底
- **突破放量**: 突破关键位置伴随放量 → 有效突破

## 综合研判框架

### 多指标共振
当多个指标同时发出相同信号时，可信度更高：

**强势买入信号**:
- MA多头排列
- MACD零轴上方金叉
- RSI在50-70区间向上
- KDJ金叉
- 成交量温和放大

**强势卖出信号**:
- MA空头排列
- MACD零轴下方死叉
- RSI在30-50区间向下
- KDJ死叉
- 成交量异常放大

### 注意事项
1. 单一指标可能产生假信号，需多指标验证
2. 不同市场环境下指标有效性不同
3. 技术分析需结合基本面和消息面
4. 历史表现不代表未来收益
