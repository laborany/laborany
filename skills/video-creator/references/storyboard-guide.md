# 分镜脚本指南

## 分镜脚本结构

分镜脚本是视频制作的蓝图，定义每个场景的内容、时间和动画。

### 基本格式

```markdown
## 场景 1: 开场
- 时间: 0:00 - 0:05 (5秒)
- 画面: Logo 居中显示
- 动画: FadeIn + ScaleIn
- 文字: 无
- 备注: 品牌色背景

## 场景 2: 问题展示
- 时间: 0:05 - 0:12 (7秒)
- 画面: 痛点场景插图
- 动画: SlideIn from-left
- 文字: "您是否遇到过这样的问题？"
- 备注: 使用问号图标强调
```

---

## 场景类型

### 1. 标题场景 (Title)

```markdown
## 标题场景
- 时间: 3-5 秒
- 元素:
  - 主标题（大字）
  - 副标题（小字，可选）
  - Logo（角落，可选）
- 动画: FadeIn / ScaleIn
- 背景: 纯色 / 渐变
```

### 2. 内容场景 (Content)

```markdown
## 内容场景
- 时间: 5-10 秒
- 元素:
  - 标题
  - 要点列表 / 段落
  - 配图（可选）
- 动画: 交错入场
- 布局: 左右分栏 / 上下结构
```

### 3. 数据场景 (Data)

```markdown
## 数据场景
- 时间: 5-15 秒
- 元素:
  - 图表（柱状图/折线图/饼图）
  - 数据标签
  - 关键数字高亮
- 动画: 渐进式揭示
- 备注: 数字滚动效果
```

### 4. 转场场景 (Transition)

```markdown
## 转场场景
- 时间: 0.5-1 秒
- 类型: Fade / Slide / Wipe
- 用途: 章节切换 / 主题转换
```

### 5. 结尾场景 (Outro)

```markdown
## 结尾场景
- 时间: 3-5 秒
- 元素:
  - CTA 文字
  - Logo
  - 联系方式（可选）
- 动画: FadeIn
```

## 时间线规划

### 短视频 (15-30秒)

```
┌─────────┬─────────┬─────────┬─────────┐
│  开场   │  核心   │  核心   │  结尾   │
│  3秒    │  10秒   │  10秒   │  3秒    │
└─────────┴─────────┴─────────┴─────────┘
```

### 中等视频 (1-3分钟)

```
┌─────┬─────┬─────┬─────┬─────┬─────┬─────┐
│开场 │问题 │方案 │详解1│详解2│详解3│结尾 │
│ 5s  │ 10s │ 15s │ 30s │ 30s │ 30s │ 10s │
└─────┴─────┴─────┴─────┴─────┴─────┴─────┘
```

### 长视频 (5分钟+)

```
┌─────┬─────┬─────────────────────────┬─────┐
│开场 │概述 │      主体内容           │结尾 │
│ 10s │ 30s │   多个章节，每章1-2分钟  │ 20s │
└─────┴─────┴─────────────────────────┴─────┘
```

## 动画标注

### 入场动画

| 标注 | 效果 | 适用场景 |
|------|------|----------|
| `FadeIn` | 淡入 | 通用 |
| `SlideIn-left` | 从左滑入 | 列表项 |
| `SlideIn-right` | 从右滑入 | 配图 |
| `SlideIn-up` | 从下滑入 | 标题 |
| `ScaleIn` | 缩放入场 | 强调元素 |
| `Typewriter` | 打字机 | 文字 |

### 退场动画

| 标注 | 效果 |
|------|------|
| `FadeOut` | 淡出 |
| `SlideOut-left` | 向左滑出 |
| `ScaleOut` | 缩小消失 |

### 转场动画

| 标注 | 效果 |
|------|------|
| `Fade` | 淡入淡出 |
| `Slide-left` | 向左滑动 |
| `Wipe` | 擦除 |
| `Flip` | 翻转 |

## 分镜脚本模板

### 产品演示模板

```markdown
# 产品演示视频分镜

## 基本信息
- 总时长: 60 秒
- 分辨率: 1920×1080
- 帧率: 30fps
- 风格: 极简现代

---

## 场景 1: 开场 Logo
- 时间: 0:00 - 0:03
- 画面: 品牌 Logo 居中
- 动画: ScaleIn + FadeIn
- 背景: 品牌主色

## 场景 2: 痛点展示
- 时间: 0:03 - 0:10
- 画面: 问题场景插图
- 文字: "还在为 XXX 烦恼？"
- 动画: SlideIn-left (图) + Typewriter (文字)

## 场景 3: 解决方案
- 时间: 0:10 - 0:18
- 画面: 产品界面截图
- 文字: "XXX 帮你轻松解决"
- 动画: FadeIn

## 场景 4-6: 功能展示
- 时间: 0:18 - 0:48
- 每个功能 10 秒
- 左侧: 功能图标 + 标题
- 右侧: 功能演示 GIF/截图

## 场景 7: CTA
- 时间: 0:48 - 0:55
- 画面: "立即体验" 按钮
- 动画: ScaleIn + 脉冲效果

## 场景 8: 结尾
- 时间: 0:55 - 1:00
- 画面: Logo + 网址
- 动画: FadeIn
```

### 数据报告模板

```markdown
# 数据报告视频分镜

## 基本信息
- 总时长: 90 秒
- 分辨率: 1920×1080
- 帧率: 30fps
- 风格: 商务专业

---

## 场景 1: 标题
- 时间: 0:00 - 0:05
- 画面: 报告标题 + 日期
- 动画: FadeIn

## 场景 2: 概述
- 时间: 0:05 - 0:15
- 画面: 3 个关键数字卡片
- 动画: 交错 ScaleIn + Counter

## 场景 3: 趋势图
- 时间: 0:15 - 0:35
- 画面: 折线图
- 动画: 线条绘制 + 数据点弹入
- 文字: 关键洞察标注

## 场景 4: 对比图
- 时间: 0:35 - 0:55
- 画面: 柱状图对比
- 动画: 柱子交错生长

## 场景 5: 分布图
- 时间: 0:55 - 1:15
- 画面: 饼图
- 动画: 扇形展开

## 场景 6: 结论
- 时间: 1:15 - 1:25
- 画面: 3 个要点总结
- 动画: 列表交错入场

## 场景 7: 结尾
- 时间: 1:25 - 1:30
- 画面: Logo + 联系方式
```

## 审核清单

在开始制作前，确认以下事项：

- [ ] 总时长是否合适？
- [ ] 每个场景时间是否充足？
- [ ] 动画节奏是否一致？
- [ ] 文字是否简洁易读？
- [ ] 视觉风格是否统一？
- [ ] 是否有明确的 CTA？
